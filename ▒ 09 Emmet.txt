■■■ Emmet(에밋) 구 - zen coding ■■■

○ 개요
- 기존에는 zen coding 이라는 이름으로 사용되었다.
- 이후 Google 에서 이 도구를 수용하면서 emmet 이라는 이름으로 변경되었다.
- HTML 생산성 도구(HTML 코딩을 빠르게 하기 위한 플러그인)
- CSS 선택자 사용(CSS3 지원)
  → HTML 을 CSS 선택자를 통해서 검색하는 방법 대신 CSS 를 통해 HTML 을 생성하도록 하는 개념.


○ 환경 설정
- 대부분의 툴에서 지원하고 있음 (※ 대부분의 편집기 내의 액션 키: Ctrl + E)

- http://emmet.io
  ① 페이지 메인에 있는 Download 버튼 클릭 → https://emmet.io/download/
  ② Eclipse/Aptana 메뉴 클릭
  ③ Installation 항목대로 진행
  ④ 플러그인 설치 완료 후 이클립스 재구동(다시 시작)
